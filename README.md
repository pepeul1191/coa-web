## CodeIgniter

Instalación de dependencias:

    $ composer install

Archivo de configuración de variables:

    application/config/config.php

Para recargar el autoload de clases:

    $ composer dump-autoload -o

Arancar servidor de prueba

    $ php -S localhost:8000
